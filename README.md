## 🛠️ Tech Stack

| Layer                            | Technology                                                 |
| -------------------------------- | ---------------------------------------------------------- |
| 🎨 **Frontend**                  | EJS (Embedded JavaScript Templates), HTML, CSS, JavaScript |
| ⚙️ **Backend**                   | Node.js, Express.js                                        |
| 🗄️ **Database**                 | MongoDB                                                    |
| 🔐 **Authentication & Security** | Crypto Libraries                                           |
| ⛓️ **Blockchain**                | Blockchain-based transaction and verification modules      |
| 🤖 **AI / ML**                   | HuggingChain for AI-powered processing and workflows       |
| 📦 **Package Management**        | npm                                                        |

## 🚀 Key Features

* Secure user authentication and data management
* MongoDB-powered scalable data storage
* AI-assisted workflows using HuggingChain
* Blockchain-backed verification and cryptographic operations
* Server-side rendering with EJS templates
* Modular Node.js backend architecture
* Interactive visual documentation for database design and application flow
